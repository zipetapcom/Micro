# Micro
A messaging plugin which allows you to do a lot of things.

Features: 
- Editable messages in-game & via config.
- Editable commands in-game & via config.
- Fully customizable.
- Open source.

Commands:
- /micro info
- /micro version
- /micro reload
- /micro test

Plugin Information:
- Author: @oliversalg
- Version: v1.0.0
- Dependencies: None
- Soft-Dependencies: None
